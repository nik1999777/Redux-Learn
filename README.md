## Usage
### Installing dependencies
To install dependencies, run the command:
```sh
$ cd redux
$ yarn
```
```sh
$ cd redux-saga
$ yarn
```
```sh
$ cd redux-ts
$ yarn
```

### Launching a development server
To start the development server, run the command:
```sh
$ cd redux
$ yarn start
```
```sh
$ cd redux-saga
$ yarn start
```
```sh
$ cd redux-ts
$ yarn start
```

### Build project
To build a project, run the command:
```sh
$ cd redux
$ yarn build
```
```sh
$ cd redux-saga
$ yarn build
```
```sh
$ cd redux-ts
$ yarn build
```
